## Ejemplos de Uso Básico

### Leer, Escribir y trabajar con Knowledge Base

Añade un texto similar al siguiente en tus instrucciones para tu agente AI:

```
Instrucciones para realizar tareas:

- Busca primero si existe alguna información relevante en tu base de conocimiento.
- Si la información no existe, realiza la tarea y guarda los resultados en la base de conocimiento así como cualquier información relevante que encuentres para futuras consultas.
- Usa la base de conocimiento para responder a las preguntas del usuario.
```

### Usar la búsqueda semántica

Añade un texto similar al siguiente en tus instrucciones para tu agente AI:

```
Usa la búsqueda híbrida de remembrances para encontrar información relevante. La búsqueda híbrida combina coincidencia literal y búsqueda semántica para obtener los mejores resultados.
```

### Almacenar y recuperar datos clave-valor

Añade un texto similar al siguiente en tus instrucciones para tu agente AI:

```
Guarda el siguiente hecho en la memoria inmediata con clave "hecho_importante": "La capital de Francia es París".
Luego, cuando necesites recuperar este hecho, usa la clave "hecho_importante" para obtener el valor almacenado.
```
