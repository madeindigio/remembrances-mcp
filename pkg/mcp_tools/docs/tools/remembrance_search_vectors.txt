TOOL: remembrance_search_vectors
=================================

Search remembrances by semantic similarity.

DESCRIPTION
-----------
Embeds the query and returns the closest stored vectors for the user.

WHEN TO CALL
------------
Use when you want results related by meaning (e.g., find notes about "budget" 
even if the note doesn't contain the word). Set "limit" to control result count.

ARGUMENTS
---------
user_id: string (required)
    The user identifier. If unsure, use the current project name.

query: string (required)
    The search query - will be embedded and matched semantically.

limit: integer (optional, default: 10)
    Maximum number of results to return.

EXAMPLE
-------
{
    "user_id": "my-project",
    "query": "follow up on project budget",
    "limit": 5
}

RELATED TOOLS
-------------
- remembrance_add_vector: Store content
- remembrance_update_vector: Modify content
- remembrance_hybrid_search: Search all layers
