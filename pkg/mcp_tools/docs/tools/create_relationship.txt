TOOL: remembrance_create_relationship
======================================

Create a relationship between two graph entities.

DESCRIPTION
-----------
Links two existing entities with a typed relationship and optional properties.
Accepts either entity names or SurrealDB record IDs for both entities.

WHEN TO CALL
------------
Use to model connections (e.g., person->works_at->organization, person->knows->person).

ARGUMENTS
---------
from_entity: string (required)
    The source entity name or ID.
    Examples: "Alice" (by name) or "entities:abc123" (by ID)

to_entity: string (required)
    The target entity name or ID.
    Examples: "Acme Corp" (by name) or "entities:def456" (by ID)

relationship_type: string (required)
    The type of relationship (works_at, knows, owns, belongs_to, etc.).

properties: object (optional)
    Additional properties for the relationship.

EXAMPLE
-------
{
    "from_entity": "Alice",
    "to_entity": "Acme Corp",
    "relationship_type": "works_at",
    "properties": { "since": "2023-01-01", "role": "Engineer" }
}

RELATED TOOLS
-------------
- remembrance_create_entity: Create entities first
- remembrance_traverse_graph: Explore the connections
