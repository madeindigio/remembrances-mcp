TOOL: remembrance_get_fact
==========================

PURPOSE
-------
Fetch a previously stored fact by exact key for a given user.

Retrieve a key-value fact for a user by key.

DESCRIPTION
-----------
Returns the stored value for the given user/key. If not found, returns nil.

WHEN TO CALL
------------
Use when you know the exact key you stored and need the precise value back 
(no semantic matching).

ARGUMENTS
---------
user_id: string (required)
    The user identifier. If unsure, use the current project name.

key: string (required)
    The key to retrieve.

EXAMPLE
-------
{
    "user_id": "my-project",
    "key": "preferred_language"
}

RELATED TOOLS
-------------
- remembrance_save_fact: Store a fact
- remembrance_list_facts: List all facts
- remembrance_delete_fact: Delete a fact
